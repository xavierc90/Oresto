Intégrer le composant Oresto

Ajoutez simplement ce composant sur votre site web pour que vos clients puissent réserver directement depuis vos pages.

Exemple d’intégration du composant :

          <script src="https://votre-lien-vers-oresto.js"></script>
          <div id="oresto-widget"></div>
          <script>
            Oresto.init('#oresto-widget', {
              restaurantId: '12345',
              // Autres paramètres si nécessaire
            });
          </script>
